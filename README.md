# Telegram anonymous message bot

Anonymous messaging bot for channels

## Features

- [x] send anonymous message
- [x] auto send to channel by confirmation
- [x] delete by sender
- [x] reply to sender's message by admin and upside down
- [ ] flexiable message
- [ ] flexiable setting

### Usage

set token

```shell
export TOKEN=TOKEN
```

run

```shell
docker-compose up
```
